# MIRAGE workflowr website

This repository hosts a workflowr website for organizing notes, toy examples, and implementation planning for the MIRAGE rare-variant association project.

Website: https://qishi7.github.io/Mirage/

## Page guide

- **Home**: project overview, purpose of the site, and a short MIRAGE paper summary.
- **Method**: technical notes on the MIRAGE model, likelihood, EM algorithm, Bayes factors, posterior probabilities, and Bayesian FDR.
- **Speed**: runtime bottlenecks and a practical optimization plan.
- **Examples**: fake toy input data and a small MIRAGE workflow template.
- **Notes**: project log, remaining questions, next steps, and reproducibility checklist.

## Data privacy

No real WES, ASD, dbGaP, SPARK, protected, private, or large data files are included in this repository. Examples use toy data only. Real or access-controlled data should remain outside the repository and should not be committed to GitHub.
