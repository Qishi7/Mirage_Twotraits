Toy Simulation Only

This page runs a small synthetic software smoke check using the current implementation. It does not use real ASD, schizophrenia, WES, dbGaP, SPARK, protected, private, or large data.

The goal is to verify that the code runs end-to-end, converges, and separates a strong synthetic pleiotropic signal from non-pleiotropic genes. This is not a formal parameter-recovery, power, or FDR-calibration study.

source("R/two_trait_mirage.R")
source("R/simulate_two_trait_mirage.R")

set.seed(20260612)
sim <- simulate_two_trait_mirage(
  n_genes = 120,
  variants_per_gene = 8,
  pi = c("00" = 0.70, "10" = 0.10, "01" = 0.08, "11" = 0.12),
  eta_trait1 = c("1" = 0.45, "2" = 0.10),
  eta_trait2 = c("1" = 0.45, "2" = 0.10),
  signal_mean_trait1 = 3.5,
  signal_mean_trait2 = 3.5,
  null_mean = -1.5,
  seed = 20260612
)

fit <- mirage_two_trait(
  sim$trait1,
  sim$trait2,
  n1_trait1 = 500,
  n0_trait1 = 500,
  n1_trait2 = 500,
  n0_trait2 = 500,
  log_bf_col_trait1 = "log_bf",
  log_bf_col_trait2 = "log_bf",
  eta_init_trait1 = c("1" = 0.20, "2" = 0.05),
  eta_init_trait2 = c("1" = 0.20, "2" = 0.05),
  pi_init = c("00" = 0.75, "10" = 0.08, "01" = 0.07, "11" = 0.10),
  max_iter = 100,
  verbose = FALSE
)
